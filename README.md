# Магазин кроссовок Vue Sneakers

## Описание

Веб приложение магазина для продажи кроссовок. Идея и дизайн проекта взяты с видео https://www.youtube.com/watch?v=U_-Ht_v-oAs

### Основные идеи

- Каталог
  - Вывод всех товаров
  - Поиск товара по названию
- Корзина
  - Добавление
  - Удаление
- Избранное
  - Добавление
  - Удаление
- Оформление заказ
  - Ввод номера телефона

### Серверная сторона

Rest API для данного приложения: https://github.com/mkstas/nest-sneakers

## Установка

### Инсталляция

```sh
npm install
```

### Режим разработки

```sh
npm run dev
```

### Сборка приложения

```sh
npm run build
```

### Предпросмотр собранного проекта

```sh
npm run preview
```

### Запуск ESLint

```sh
npm run lint
```

### Форматирование файлов

```sh
npm run format
```
