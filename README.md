# Vue Sneakers

Приложение магазина по продаже кроссовок

### Установка проекта

```sh
npm install
```

### Запуск json-server

```sh
npm run db
```

### Сборка для разработки

```sh
npm run dev
```

### Сборка готового проекта

```sh
npm run build
```

### Предпросмотр готового проекта

```sh
npm run preview
```
